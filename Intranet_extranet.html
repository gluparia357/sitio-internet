Intranet

Una intranet es una red privada que utiliza tecnología de Internet para compartir información dentro de una organización. Se utiliza para mejorar la comunicación interna y la colaboración.

Características:

  • Acceso restringido solo a personal autorizado.
  • Facilita la comunicación y el trabajo en equipo.
  • Proporciona acceso a recursos internos, como documentos y aplicaciones.

Extranet

Una extranet es una extensión de una intranet que permite el acceso a ciertos datos y recursos a personas externas, como socios o proveedores, a través de una conexión segura.

Características:

  • Acceso controlado para usuarios externos.
  • Facilita la colaboración con socios comerciales.
  • Mejora la comunicación y la eficiencia en los procesos de negocio.
